- Collapse: `Ctrl + Shift + [` 
- Expand: `Ctrl + Shift + ]`
- Collapse all: `Ctrl + K, Ctrl + 0`
- Expand all: `Ctrl + K, Ctrl + J`

Rodar localmente:
  - npm i
  - npm start
